# Códigos ANSI para Colores en Java

## Colores de Texto

| Color          | Código ANSI |
|----------------|-------------|
| **Negro**      | `\u001B[30m` |
| **Rojo**       | `\u001B[31m` |
| **Verde**      | `\u001B[32m` |
| **Amarillo**   | `\u001B[33m` |
| **Azul**       | `\u001B[34m` |
| **Morado**     | `\u001B[35m` |
| **Cian**       | `\u001B[36m` |
| **Blanco**     | `\u001B[37m` |
| **Reset**      | `\u001B[0m`  |
